# GSC SEO Sprint Notes

Date: 2026-06-23

## Context

- Google Search Console showed low overall CTR with most visibility sitting on prison entity pages, prison intent pages, and selected UK browse hubs.
- The clearest near-term opportunities were prison pages already receiving impressions, plus intent pages already ranking around page one or page two for visit-related searches.

## Changes Implemented

- Updated prison profile title generation to use more query-aligned patterns for prison entity searches.
- Added targeted title overrides for:
  - `hmp-wandsworth`
  - `hmp-leicester`
  - `hmp-manchester`
  - `florence-admax-usp`
- Updated prison profile meta descriptions to include search-facing entity phrasing plus visiting/contact/address context.
- Updated prison intent page titles and descriptions to better match common search patterns such as:
  - visiting times
  - contact details
  - booking a visit
  - what to expect
- Strengthened country-level metadata for:
  - `/prisons/uk`
  - `/prisons/us`
- Strengthened metadata and intro copy for:
  - UK security category hubs
  - UK subgroup hubs
- Tightened UK collection meta descriptions to use clearer browse-oriented phrasing.

## Priority Targets

- `/prisons/uk/hmp-wandsworth`
- `/prisons/uk/hmp-leicester`
- `/prisons/uk/hmp-manchester`
- `/prisons/us/florence-admax-usp`
- `/prisons/uk`
- `/prisons/uk/category/category-c`
- `/prisons/uk/subgroup/long-term-and-high-security-estate`

## Follow-up

- Review GSC again after reindexing to compare CTR and average position changes on the target URLs.
- Decide whether legacy root-level article URLs on the domain should remain live, be redirected, or be consolidated into the Next.js content architecture.
- Consider a second pass on top-performing prison intent pages if visit-related queries continue to grow.
